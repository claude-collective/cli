# Release 0.109.0 (2026-04-06)

## Added

- **Eject indicator pill in skill tags**: ejected skills now show a `⏏` pill with background between the scope badge and skill name in the build step, making source type visible at a glance.

## Fixed

- **Scope toggle guard for ejected skills**: toggling a project-scoped ejected skill to global scope is now blocked with a toast message when a global ejected copy already exists, preventing file overwrites that would affect other projects.
