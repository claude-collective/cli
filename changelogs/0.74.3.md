# Release 0.74.3 (2026-03-13)

## Fixed

- **validate**: Enable verbose logging for source validation — `--verbose` now correctly shows slug resolution messages during `cc validate --source`
- **init**: Resolve symlink paths before comparing project and global install directories — fixes macOS `/var` → `/private/var` issue where `.claude-src/` was created before the wizard rendered
- **config-types**: Forward extra categories through the project-import generation path — custom categories were silently dropped when a global installation existed

## Changed

- **matrix**: Demote "Unresolved slug" messages from `warn()` to `verbose()` — these diagnostic messages now only appear with `--verbose`, reducing noise during normal operation

## Tests

- Fix E2E source fixtures with required metadata fields (`cliDescription`, `usageGuidance`, `displayName`)
- Fix `contentHash` values to match `^[a-f0-9]{7}$` validation pattern
- Replace all fake SkillIds in E2E tests with valid ones from the generated union type
- Remove environment-dependent config-types comment assertions
