# Release 0.96.0 (2026-03-28)

## Changed

- Renamed "local" install mode to "eject" throughout — `InstallMode` type, `SkillConfig.source` value, all UI labels, CLI commands, configuration, JSON schema, and tests (D-156)
- Removed lock icon from agents step — toast already provides sufficient feedback for locked global agents
- Updated `UI_MESSAGES.GLOBALLY_INSTALLED` toast from "modify from ~/" to "modify from home dir"
- Updated `UI_MESSAGES.GLOBALLY_LOCKED_CATEGORY` toast from "modify from ~/" to "modify from home dir"
- `SOURCE_DISPLAY_NAMES` key renamed from `local` to `eject`
