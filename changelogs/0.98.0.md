# Release 0.98.0 (2026-03-29)

## Added

- **SkillAgentSummary component**: extracted shared skill/agent summary rendering from step-confirm into a reusable pure-content component (`skill-agent-summary.tsx`)
- **Info panel redesign**: full skill/agent summary with marketplace name, stack name, and scrollable content replaces the old stats-based view; enabled via `INFO_PANEL` feature flag (now on by default)
- **Confirm step scrolling**: confirm step uses the standard 4-level scroll pattern so large skill/agent lists are navigable with arrow keys
- **Terminal clear after wizard**: both `init` and `edit` commands call `clear()` and `clearTerminalOutput()` after the wizard exits, preventing the wizard UI from bleeding into install log output
- **Installed config snapshots**: `installedSkillConfigs` and `installedAgentConfigs` stored in wizard store for diff rendering (bullet vs +/- markers)

## Changed

- **Info panel**: replaced stats counters with scrollable SkillAgentSummary showing all skills/agents grouped by scope
- **Wizard layout**: removed `<Static>` startup messages block — "Loaded N skills" and "Found N installed skills" no longer render on top of the wizard, freeing viewport height

## Removed

- **stats-panel.tsx**: deleted — superseded by the redesigned info panel with SkillAgentSummary
- **previousSkillConfigs / previousAgentConfigs props**: removed from StepConfirm — diff state now read from wizard store
