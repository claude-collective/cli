# Release 0.55.1 (2026-03-01)

## Fixed

- **Uninstall agent removal (B-08):** `agentsinc uninstall` now selectively removes only agents listed in the config's `agents` array instead of removing the entire `.claude/agents/` directory. User-created agent files not listed in config are preserved.

## Changed

- Refactored `removeLocalFiles` in uninstall command into flat declarative methods (`removeMatchingSkills`, `removeMatchingAgents`, `removeEmptyClaudeDir`) with early returns instead of deeply nested branches.
- Added `configuredAgents` field to `UninstallTarget`, resolved at detection time from the generated config's `agents[]` array (preserved by schema `.passthrough()`).
- Made `collectConfiguredSources` declarative (spread-based instead of imperative push).

## Renamed

- **Subcategory → Category (D-70 Part 4):** Renamed the `Subcategory` type to `Category` across the entire codebase (~64 files). The type represents the same concept as `category` in `metadata.yaml` — the old name was misleading. Includes types, schemas, constants, functions, variables, tests, E2E, agent docs, project docs, and changelogs. `CategoryPath` unchanged (broader type that adds `"local"` and prefix forms).

## Docs

- Archived completed tasks from TODO.md: B-07, B-08, D-50, D-68, D-33, D-60.
- Removed deferred tasks already in TODO-deferred.md: D-47, D-08, D-40.
- Renamed all "subcategory" references to "category" in documentation, TODO files, and agent templates.
