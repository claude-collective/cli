# Release 0.76.0 (2026-03-16)

## Changed

- **Shared categories consolidated into `shared-meta`** — `shared-methodology`, `shared-research`, and `shared-reviewing` merged into a single `shared-meta` category (displayName: "Meta"). Reduces wizard noise and reflects that these skills operate as a unified meta layer rather than separate concerns.
- **Skill IDs updated** — `meta-reviewing-reviewing` → `shared-meta-reviewing`, `meta-reviewing-cli-reviewing` → `shared-meta-cli-reviewing`. All stack assignments, test fixtures, and E2E helpers updated accordingly.
- **All stacks updated** — `shared-methodology` and `shared-reviewing` category assignments replaced with `shared-meta` across all 9 default stacks and all agent definitions within each stack.

## Added

- **`web-tooling` category** — New category for frontend build tools and bundlers (Vite, Webpack). Previously Vite had no category home; it now lives under `web-tooling` in the web domain. All stacks that include Vite updated to assign it to this category.
- **`shared-tooling` constant in default stacks** — ESLint/Prettier, TypeScript config, and git hooks extracted into a shared `SHARED_TOOLING` constant, replacing the previous `SHARED_METHODOLOGY` constant across all stack definitions.
