# Release 0.68.0 (2026-03-11)

## Added

- **Per-agent scope in edit mode (D-74)** — edit command now detects agent scope changes (project/global) and displays them in the change summary alongside skill scope changes
- Agent scope configs are restored from installed config when entering edit mode, preserving previous project/global assignments
- Help modal now shows `S` key hint for agent scope toggle on the agents step
- Integration tests for per-agent scope: default scope, toggle, roundtrip persistence, edit-mode restoration

## Changed

- Edit command's "no changes" detection now considers agent scope changes in addition to skill changes
- Scope change summary in edit output combines skill and agent scope counts
