# Release 0.54.0 (2026-03-01)

## Changed

- **Renamed `ts-config-*` to `config-*`** — Configuration files (`ts-config-loader.ts`, `ts-config-types-writer.ts`, `ts-config-writer.ts`) and their tests renamed to drop the `ts-` prefix for consistency, since all configs are now TypeScript.

## Fixed

- **Domain ordering mismatch in wizard** — Custom domains now appear first (alphabetically) in all wizard views. Previously `sortDomainsCanonically()` and the domain selection checkbox list put built-in domains first while the tab bar put custom domains first, causing the highlighted tab to not match the first visible tab on edit. Domain selection now delegates to `orderDomains()` instead of duplicating ordering logic.
