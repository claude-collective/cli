# Release 0.63.0 (2026-03-09)

## Changed

- **Compile: remove `--output` flag** — Compile always targets the Claude plugin directory. Removed `runCustomOutputCompile`, associated message constants, and all `--output` tests.
- **Wizard: rename Build step to Skills** — Tab label changed from "Build" to "Skills" across all wizard components.
- **Wizard: always show scope badge** — Every skill in the skills step now displays `[P]` (project) or `[G]` (global) badge, not just after toggling.
- **Wizard: centralize hotkeys** — New `hotkeys.ts` registry with `HOTKEY_*` constants, `KEY_LABEL_*` display strings, and `isHotkey()` helper. 10 wizard components updated to import from registry instead of hardcoding key strings.
- **Wizard: remove `label` from `SourceOption`** — Removed `label` prop and `formatSourceLabel()` from wizard store. Display name now derived at render time in `source-grid.tsx` from source ID + installed flag.
- **Matrix store: replace direct `matrix.skills[id]` with store accessors** — 7 production files migrated from `matrix.skills[id]` to `findSkill()`/`getSkill()` from matrix store. Fixed unsafe `!` assertions in `getDiscourageReason` with graceful `findSkill` fallback.
- **Domain selection: remove `matrix!` assertion** — Replaced `s.matrix!` with `s.getMatrix()` throwing accessor. Removed dead `?? fallback` on `BUILT_IN_DOMAIN_DESCRIPTIONS` lookup.
- **Tests: flatten default-stacks.test.ts** — Pre-computed flat array of `(stackId, agentName, category)` tuples via `flatMap` + `typedEntries`, passed to `it.each` for individual parameterized test cases.

## Removed

- `--output` flag from `compile` command
- `label` prop from `SourceOption` type
- `formatSourceLabel()` from `wizard-store.ts`
- `CUSTOM_COMPILE_COMPLETE` and `NO_AGENTS_TO_COMPILE` message constants
- `type-cleanup-tasks.md` (all tasks completed)
