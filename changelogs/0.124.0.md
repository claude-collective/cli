# Release 0.124.0 (2026-04-13)

## Fixed

- **Exclusive category deselect guard** — Skills in non-required single-skill categories (e.g., Cli Prompts with only Clack) can now be deselected. Previously the guard blocked deselection in ANY category with only one skill in the matrix. The guard now only fires when the category is both `exclusive: true` AND `required: true` (e.g., Framework where you must pick exactly one).
- **SkillTag lock icon layout** — The lock icon (🔒) on globally-installed skills no longer wraps to a new line when scope badges are present. Fixed by adding `flexDirection="row"` to the SkillTag Box and rendering the lock as a separate Text element for correct terminal width calculation.
