# Release 0.81.0 (2026-03-21)

## Added

- **Dedicated domain selection step (D-120):** Domain selection is now its own wizard step between Stack and Build. New 6-step flow: Stack → Domains → Build → Sources → Agents → Confirm. Edit mode bypasses both Stack and Domains.
- **Convention-keeper meta agent:** New agent for maintaining codebase conventions and standards.
- **Scribe meta agent:** Replaces documentor agent with improved documentation capabilities.
- **D-124 default source test:** `it.fails` test documenting D-123 ENOENT bug with accurate assertions for expected correct behavior.

## Changed

- **Wizard tabs no longer show step numbers (D-121):** Tabs display as `Stack  Domains  Skills  Sources  Agents  Confirm` instead of `[1] Stack  [2] Skills  ...`
- **EditWizard POM detection:** Now waits for category counter text instead of "Framework" to avoid matching stack step's "Other Frameworks" group label.
- **Updated default stacks and rules:** Latest skill data, regenerated types.

## Fixed

- **Edit wizard premature text match:** EditWizard.launch() matched "Framework" in the stack step's "Other Frameworks" instead of the build step's "Framework (1 of 1)".
- **step-agents.test.tsx:** Updated "Documentor" → "Scribe" to match agent rename.

## Removed

- **Documentor agent:** Replaced by Scribe agent with updated workflow.
- **Tab number prefixes:** `[1]`, `[2]`, `[3]`, `[4]`, `[5]` removed from wizard tabs.
