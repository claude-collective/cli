# Release 0.110.2 (2026-04-06)

## Fixed

- Scope toggle (S hotkey) no longer blocked by excluded override entries — `toggleSkillScope` guards now filter `excluded: true` entries, allowing users to move a project-scoped skill back to global
- Agent cross-contamination in stack property — `buildEjectConfig` no longer assigns all skills to all agents; uses per-agent assignments from the stack definition filtered to user-selected skills
- Two dual-scope edit E2E tests converted from `it.fails` to passing tests
