# Release 0.89.0 (2026-03-27)

## Added

- 6 missing category definitions in `default-categories.ts`: `api-cms`, `api-commerce`, `api-framework`, `api-search`, `api-vector-db`, `meta-design`
- Exhaustive type check on `defaultCategories` using `as const satisfies Record<Category, CategoryDefinition>` — new categories in the generated union now cause a compile error until added
- Wizard history seeding on initialization — ESC from the build step now navigates back through prior steps (domains, stack) instead of jumping to stack

## Changed

- Meta domain display name in wizard: "Methodology" to "Meta" (now covers design patterns, code review, and research methodology)
- Meta domain description updated to "Design patterns, code review, and research methodology"

## Fixed

- ESC from build step in edit mode skipped domain selection (empty history fallback). Now walks through `WIZARD_STEPS` via `setStep()` during initialization so history builds naturally.
