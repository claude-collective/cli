# Release 0.97.0 (2026-03-29)

## Added

- **UI constants**: `UI_SYMBOLS.EJECT` (⏏ U+23CF), `UI_SYMBOLS.BULLET` (• U+2022), `CLI_COLORS.LABEL_BG` (#383838) — shared across wizard steps
- **Confirm step redesign**: two-column layout with a Skills box (left) and Agents box (right), each flex-growing to 50% width
  - Scope sections (Project / Global) styled with `LABEL_BG` background label, matching the build step category headings
  - Skills render in 2 columns with `flexWrap`; agents render 1 per line
  - Existing items prefixed with `•`, new items with `+` (green), removed items with `-` (red)
  - Eject-mode skills show the ⏏ icon in place of `[eject]` text
- **Focused label style**: `LABEL_BG` background highlight on focused section headers across all wizard steps
  - Build step (category-grid): already had this; now uses `CLI_COLORS.LABEL_BG` constant instead of hardcoded hex
  - Sources step (source-grid): skill name header highlights on row focus, plain gray when unfocused
  - Agents step (step-agents): group header (Web, API, CLI, Meta) highlights when any agent in that group is focused
- **Init dashboard**: Agents Inc ASCII logo replaces plain text name; viewport fills terminal height (clear-screen effect); nav footer removed

## Changed

- `category-grid.tsx`: hardcoded `"#383838"` replaced with `CLI_COLORS.LABEL_BG`

## Added (tests)

- `step-confirm.test.tsx`: 24 unit tests covering scope labels, eject icon, bullet/+/- markers, keyboard navigation, and empty state
