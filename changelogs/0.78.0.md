# Release 0.78.0 (2026-03-20)

## Added

- **Wizard tab navigation**: step titles lifted from individual components into `wizard-layout`, with domain tab bar and dropdown selectors for quick domain switching during the build step
- **Stats panel**: build step now shows skill/scope counts (total skills, global vs project breakdown, agent count) via new `StatsPanel` component
- **`setCurrentDomainIndex` store action**: allows direct domain navigation from the tab bar
- **E2e global teardown**: removes stale `e2e-test-*` marketplaces after test suite runs to prevent accumulation
- **`claudePluginMarketplaceRemove` utility**: new exec helper for removing marketplace registrations

## Changed

- **ViewTitle removed from step components**: all step titles now rendered centrally by `wizard-layout` instead of each step component rendering its own title
- **`populateFromStack` domain selection**: now only includes domains present in the stack's actual skill selections, not all domains from all categories
- **Category grid mount focus**: wrapped in `useEffect` to fix React warnings about side effects during render
- **Global scope badge coloring**: global scope badges in category grid now use warning color to visually distinguish from project scope

## Fixed

- **React render side effect warning**: mount-time focus initialization in `CategoryGrid` moved from component body to `useEffect`
