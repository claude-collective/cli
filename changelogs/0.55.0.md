# Release 0.55.0 (2026-03-01)

## Removed

- **`--dry-run` flag removed from all commands** — The flag added ~143 lines of conditional preview logic across 6 commands (`compile`, `init`, `uninstall`, `import skill`, `new marketplace`, `new skill`), plus 30 test cases. The wizard's confirm step already previews what will be installed, and all operations are local/reversible, making `--dry-run` redundant. 19 of 22 commands inherited the flag but never implemented it. The `DRY_RUN_MESSAGES` constants and `dryRunLocalRemoval()` method were also removed.
