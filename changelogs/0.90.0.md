# Release 0.90.0 (2026-03-27)

## Added

- Dynamic marketplace display name — wizard Sources step now reads `owner.name` from `marketplace.json` instead of using a hardcoded label map. Custom marketplaces show their own name (e.g., "Photoroom" instead of "Agents Inc").
- `displayName` field on `SkillSource` type, threaded from `SourceLoadResult.marketplaceDisplayName` through `multi-source-loader` to `source-grid.tsx`

## Fixed

- macOS `/var` → `/private/var` symlink path mismatch in `detectBothInstallations` and `discoverInstalledSkills`. Used `fs.realpathSync()` for homedir comparisons — matches the pattern already used in `init.tsx` and `local-installer.ts`. This caused incorrect dual-scope filtering when `HOME` pointed to a temp dir under `/var/folders/`.
- Ctrl+C cancellation e2e test — removed `.fails` modifier and relaxed exit code assertion since cancellation timing is non-deterministic.
