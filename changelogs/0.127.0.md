# Release 0.127.0 (2026-04-14)

Wizard sources-step UX polish: global skills in project scope are now focusable with an explanatory toast, and the lock icon sits before the skill name for cleaner alignment.

## Changed

- Sources step: read-only global rows are now focusable from project scope. Pressing Space on a locked row surfaces an explanatory toast instead of being silently ignored — the user always gets feedback.
- Sources step: lock icon (🔒) moved before the skill name in both focused and non-focused render paths. Matches the visual weight of the label column and stays visible when the row is focused with the highlighted background.
- Mode-migrator tests: isolated from the real user home directory by stubbing `HOME` in the test harness. Eliminates cross-test leakage when running on a developer machine.
