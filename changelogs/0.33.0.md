- **Package renamed**: `@claude-collective/cli` → `@agents-inc/cli`
- **CLI binary renamed**: `cc` → `agentsinc`
- **Default skills source**: `github:claude-collective/skills` → `github:agents-inc/skills`

- Update schema URLs from `claude-collective/cli` to `agents-inc/cli` across all 18 agent YAML files, 3 config YAML files
- Update all CLI help text and user-facing messages to reference `agentsinc` command
- Update branding constant from "Claude Collective" to "Agents Inc."
- Update `DEFAULT_PLUGIN_NAME` and cache directory to `agents-inc`
- Update agent markdown content (`agent-summoner`, `skill-summoner`) to reference `agentsinc` commands
- Update documentation (`commands.md`, `architecture-bible.md`, `data-models.md`, proposed features)
- Update README with new package name, binary, and installation instructions
- Regenerate `package-lock.json` and `bun.lock` with new package name
