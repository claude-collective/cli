# Release 0.74.4 (2026-03-13)

## Fixed

- **stacks**: Validate custom skill IDs via matrix instead of broad pattern — `resolveAgentConfigToSkills` now uses `SKILL_ID_PATTERN` for standard skills and falls back to `getCustomSkillIds()` for known custom skills, instead of the overly permissive `KEBAB_CASE_PATTERN`
- **hooks**: Guard against undefined slug lookup in source grid search modal — prevents TypeScript errors when a skill ID has no corresponding slug

## Changed

- **matrix-provider**: Add `getCustomSkillIds()` getter that returns IDs of all custom skills in the current matrix
