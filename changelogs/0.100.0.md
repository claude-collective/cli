# Release 0.100.0 (2026-04-01)

## Added

- `list` command now renders installed skills and agents using the SkillAgentSummary Ink component, matching the info panel visual style (falls back to plain text for non-TTY)
- Source grid column headers ("Local", "Plugin") above source options in the sources step
- Chevron focus indicator in source grid, replacing bordered tag boxes
- Vertical navigation in source grid preserves column index instead of resetting to 0

## Changed

- Source grid layout: skill names render inline with source tags instead of above them
- Source grid options use plain text with dim/bold styling instead of bordered boxes
- Edit command post-wizard output now uses chalk colors: green for added, red for removed, yellow for changed, gray for operational messages
- Edit command collapses install/uninstall progress to single summary lines
- Edit command removes noisy intermediate messages (marketplace registration, agent partials loading)
- Edit command completion simplified from verbose count summary to "Done"

## Fixed

- Source grid column persistence: moving up/down no longer resets focus to column 0
