# Release 0.110.0 (2026-04-06)

## Fixed

- **Global config immutability**: project-level operations (init, edit) no longer modify the global config's skills or agents arrays. Previously, tombstone pruning incorrectly removed entries from the global config when skills were toggled to project scope.
- **Inlined config deduplication**: when generating the inlined project config, active global entries are now replaced by their excluded tombstones instead of appearing alongside them, preventing triple duplication.
- **Scope-aware orphan detection**: `cc doctor` now checks project agent files against active project-scoped agents and global agent files against all global-scoped agents (including excluded ones that serve other projects).

## Added

- **Unit tests for agent compound key merge**: 4 tests covering dual active+excluded entries, merge with existing excluded, deduplication, and scope override.
- **E2E exclusion lifecycle test**: verifies agent file placement at correct scope, global config immutability, scope badge persistence through edit passthrough, and no duplicate agent files.
