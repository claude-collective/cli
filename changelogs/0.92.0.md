# Release 0.92.0 (2026-03-27)

## Added

- **Info panel feature flag** (`INFO_PANEL: false`) — gated replacement for the help modal, activated with the `I` key
- `info-panel.tsx` component (renders when feature flag is enabled)

## Changed

- **Wizard hotkeys**: `HOTKEY_HELP` (`?`) replaced with `HOTKEY_INFO` (`I`)
- **Wizard store**: `showHelp`/`toggleHelp` renamed to `showInfo`/`toggleInfo`
- **step-build**: removed inline stats panel and `computeStats` function
- **wizard-layout**: info panel and footer hint gated behind `FEATURE_FLAGS.INFO_PANEL`
- Formatting fixes in edit, uninstall, and multi-source-loader (line wrapping)

## Removed

- `help-modal.tsx` — replaced by info panel
- E2E help modal test and `openHelp`/`closeHelp` page object methods
- Stats panel assertions from E2E edit wizard detection tests
