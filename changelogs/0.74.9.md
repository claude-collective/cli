# Release 0.74.9 (2026-03-14)

## Fixed

- **ViewTitle consistency (D-95)**: All 7 wizard step components now use the shared `ViewTitle` component. `step-refine.tsx` was the only step missing it.
- **Logo only on first screen (D-72)**: ASCII logo now only renders on the stack selection step (first wizard screen), not on subsequent steps. Already removed from edit command in prior release.
- **Fake E2E tests relocated**: 2 unit test blocks (`writeScopedConfigs`, `splitConfigByScope`) moved from `unified-config-view.e2e.test.ts` to their proper unit test files. 3 Claude CLI smoke tests moved to `e2e/smoke/` with `.smoke.test.ts` extension.
- **Unnecessary type casts removed**: Removed `as SkillId` casts on valid union members in test files. Removed `export` from `InstallPaths` type (internal-only).

## Changed

- **R-08: Unified resolve functions**: Replaced 5 separate relationship resolution functions (`resolveConflicts`, `resolveDiscourages`, `resolveCompatibilityGroups`, `resolveRequirements`, `resolveAlternatives`) with a single `resolveRelationships` function in `skill-resolution.ts`. Net -27 lines, same behavior.
