# Release 0.114.0 (2026-04-09)

## Fixed

- **Unique skill guard**: Prevent deselecting the only skill in a category. If a category has only one skill in the matrix, the toggle is blocked with a toast message "Cannot deselect the only skill in this category".
- **Exclusive category global guard bypass**: Selecting a different skill in an exclusive category (e.g., client-state) could implicitly deselect a globally-installed skill, bypassing the direct global toggle guard. Now blocked with the same "Global skills cannot be changed from project scope" toast.

## Added

- Unit tests for both guards in `wizard-store.test.ts` (6 tests)
- E2E test for unique skill guard (`edit-wizard-unique-skill-guard.e2e.test.ts`, 2 tests)
- E2E test for exclusive category bypass guard (`global-skill-toggle-guard.e2e.test.ts`, 1 test)
