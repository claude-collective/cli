# Release 0.74.7 (2026-03-13)

## Added

- **E2E test suite expansion**: 25 new E2E test files covering commands, interactive wizards, lifecycle flows, and integration scenarios (485 total E2E tests across 56 files)
- **Shared E2E helpers**: `plugin-assertions.ts` with `verifyAgentCompiled`, `verifyConfig`, `verifyPluginInSettings`, `verifySkillCopiedLocally`, `verifyNoLocalSkills`, `verifyNoPlugins`
- **Shared E2E infrastructure**: `create-e2e-plugin-source.ts` for plugin build + marketplace setup; `waitForRawText`, `navigateInitWizardToCompletion`, `passThroughAllBuildDomains`, `getEjectedTemplatePath` helpers in `test-utils.ts`
- **Shared timing constants**: `PLUGIN_INSTALL_TIMEOUT_MS`, `EXIT_WAIT_TIMEOUT_MS`, `SETUP_TIMEOUT_MS`, `LIFECYCLE_TEST_TIMEOUT_MS`, `INTERACTIVE_TEST_TIMEOUT_MS`
- **Shared test constants**: `COMPILE_ENV`, `FORKED_FROM_METADATA` extracted from duplicated local definitions
- **E2E test specs and audit**: framework design, lifecycle test design, dual-scope edit spec, gap analysis, and comprehensive audit findings

## Changed

- **Unit test fixtures refactored**: Skill and agent test fixtures extracted from `create-test-source.ts` into dedicated `mock-skills.ts` and `mock-agents.ts` modules
- **E2E code quality audit applied**: Removed task IDs from describe blocks, unnecessary type casts, `undefined!` non-null assertions, duplicated constants, raw `readFile`/`readdir` calls, magic timeout numbers, and manual checks replaced with shared assertion helpers
