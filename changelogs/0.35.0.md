# 0.35.0

## Selective Uninstall

- Rework `uninstall` command to selectively remove only CLI-installed skills
- Skills with `generatedByAgentsInc: true` in metadata.yaml are removed; user-created skills are preserved
- Inject `generatedByAgentsInc: true` into metadata.yaml during skill installation
- Add `readLocalSkillMetadata()` for reading full local skill metadata
- Remove `.claude/` only if empty after selective cleanup
- Add dry-run preview for selective removal

## Expert Mode Persistence

- Persist `expertMode` to project config (config.yaml) across sessions
- Read saved `installMode` and `expertMode` from config in edit command
- Default `expertMode` to `true` when `--source` flag is passed in init
- Add `expertMode` field to project config Zod schemas and TypeScript types

## Compile Metadata Requirement

- Require `metadata.yaml` for local skill discovery during compilation
- Skip skills without metadata.yaml and emit a warning with skill name and path

## Installer Fix

- Preserve `preloaded: true` flags from stack YAML definitions onto generated project config
- Add config option hints comment to generated config.yaml for discoverability

## Wizard Improvements

- Add pixel-accurate `measureElement`-based scroll viewport to source grid
- Use Ink `isActive` prop for conditional input handling (replaces manual early-return guards)
- Always display marketplace label with fallback to default plugin name

## Skills Matrix

- Make styling category non-exclusive to allow CVA alongside SCSS Modules / Tailwind
