# Release 0.71.0 (2026-03-11)

## Added

- **Reusable `SelectList<T>` component** (`src/cli/components/common/select-list.tsx`) — vertical single-select list with `❯` cursor, keyboard navigation via `useKeyboardNavigation`, optional `renderItem` prop for custom row rendering
- **`clearTerminalOutput()`** helper in `init.tsx` — clears the visible terminal between successive Ink render instances so views don't stack

## Changed

- **Dashboard** — removed stats display (skills, agents, mode, source); simplified to branding + vertical SelectList menu with Edit, Compile, Doctor, List options
- **GlobalConfigPrompt** — removed "What would you like to do?" text; replaced horizontal button bar with vertical SelectList
- **SearchModal** — replaced custom `ResultRow` component and manual `useKeyboardNavigation` with `SelectList` using custom `renderItem`; marker changed from `▸` to `❯` (`UI_SYMBOLS.CHEVRON`)
