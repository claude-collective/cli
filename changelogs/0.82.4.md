# Release 0.82.4 (2026-03-24)

- `splitConfigByScope` now correctly splits global agents' stack between global and project configs when skills have mixed scope — project-scoped skills under global agents were silently dropped
- `cc edit` now detects added/removed agents — previously exited with "No changes made" when only agents changed
- `cc edit` now copies newly added local-source skills to `.claude/skills/` split by scope
