# Release 0.72.0 (2026-03-12)

## Added

- Codegen script (`generate:types`) that scans the skills source and generates TypeScript types for SkillSlug, SkillId, Category, Domain, and AgentName
- Generated `src/cli/types/generated/source-types.ts` as single source of truth for all type unions

## Changed

- SkillSlug, SkillId, Category, Domain, and AgentName unions are now derived from the generated file instead of manually maintained
- Zod schema validators use generated arrays for `z.enum()`, enforcing strict slug validation at parse boundaries
- Default rules and stacks updated to use slug-based references consistent with generated types

## Removed

- ~250 lines of manually-maintained union members from `skills.ts`, `matrix.ts`, and `agents.ts`
- Orphan categories with no matching skills in the source: `web-base-framework`, `mobile-platform`, `cli-prompts`, `cli-testing`
- Redundant init flag validation tests and broken search interactive mode test
