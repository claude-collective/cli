# 0.32.0

## Features

- **Wizard simplification**: Remove step-approach; stack vs scratch choice merged into step-stack with single "Start" button
- **Configurable branding**: Add `DEFAULT_BRANDING` constant; replace all hardcoded "Claude Collective" strings
- **Scroll viewport**: Add `useMeasuredHeight`, `useTerminalDimensions`, and `useVirtualScroll` hooks for dynamic category grid scrolling
- **agents_dir config override**: Support custom agents directory in project source configuration
- **YAML-driven agent-skill mappings**: `agent_skill_prefixes` loaded from `agent-mappings.yaml` with hardcoded fallback

## Improvements

- Extract `STACKS_FILE_PATH` and `DEFAULT_SCRATCH_DOMAINS` to shared constants
- `formatStepLabel` returns structured `{prefix, label, full}` object
- Move `TEST_AVAILABLE_SKILLS` to shared test constants
- Fix `runCliCommand` to intercept both `process.stdout.write` and `console.log` (bun compatibility)
- Remove unnecessary comments across production and test code
