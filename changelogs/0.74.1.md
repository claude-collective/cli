# Release 0.74.1 (2026-03-13)

## Fixed

- **Uninstall scope for global plugins**: Global plugins installed with `user` scope were not uninstalled because `claudePluginUninstall` hardcoded `project` scope. Now dynamically determines scope based on install directory.
- **Plugin install marketplace gate**: `BUILT_IN_MATRIX` optimization skipped `marketplace.json` fetch, leaving `sourceResult.marketplace` undefined and forcing an incorrect fallback to local mode. Removed the gate and added lazy marketplace resolution in `installIndividualPlugins`.
- **Edit-global flag forwarding**: Selecting "Edit global installation" from the init prompt now forwards `--source` and `--refresh` flags to the edit command, instead of silently falling back to the default remote source.
