# Release 0.107.0 (2026-04-06)

## Added

- Scope-grouped source grid (D-184) — sources step separates skills into "Global" and "Project" sections with inline left-column labels when both scopes are present; flat rendering when only one scope exists
- Toast message system — `toastMessage` store state with auto-clear timer, absolute-positioned toast overlay in wizard layout
- Global scope guard (D-187) — pressing S (scope toggle) in build or agents step shows "Scope toggle unavailable in global context" toast when running from HOME directory
- `isGlobalRoot` detection in init command — computed once and threaded through to wizard as `isEditingFromGlobalScope`
- `pretest:e2e` script in package.json — auto-builds before E2E test runs
- E2E test for scope toggle interaction — verifies S hotkey toggles skill scope from global to project

## Changed

- Source grid renders scope labels ("Global" / "Project") as a left column on the first row of each group, replacing the previous full-width separator rows
- Toast component uses `position="absolute"` for overlay rendering
- `buildSourceRows()` now includes `scope` from `skillConfigs` on each row
- `ensureGlobalConfig` accepts `isGlobalRoot` boolean instead of computing it internally
