# Release 0.42.0 (2026-02-19)

## Added

- `eject agent-partials --templates` flag to eject only agent templates instead of the full partials directory (includes `-t` shorthand)
- Reusable `SelectionCard` component for wizard card-style selections (rounded border, focus state, label + description)
- `CheckboxGrid` now accepts optional `renderItem` prop for custom item rendering
- `CLI_COLORS.WHITE` constant
- TODO tracking for B-02 (metadata ref validation) and B-03 (edit pre-selection compatibility)

## Changed

- Wizard tabs: active tab now uses background color with bold text and centered alignment
- `ViewTitle`: simplified to bold warning-colored text (removed background blocks)
- Category grid: removed dim borders on unfocused skills, focused category name uses white color
- Build step: active domain tab uses warning color instead of primary
- Confirm step: "verified" label uses primary color instead of success
- Help modal: improved spacing between key bindings
- `CheckboxGrid`: uses chevron pointer instead of filled circle, shows description only when focused, subtitle is optional
- Stack selection uses `SelectionCard` instead of `MenuItem`
- Sources step uses `SelectionCard` instead of inline card boxes
- Step refine: wrapped long branding text into single `<Text>` element

## Fixed

- Edit command: fall back to project config skills when plugin discovery returns empty (local mode), ensuring correct pre-selection in edit wizard
