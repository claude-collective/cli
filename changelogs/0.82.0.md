# Release 0.82.0 (2026-03-23)

## Added

- Three new domains: `ai` (20 skills), `meta` (3 skills), `infra` (4 skills)
- AI domain categories: `ai-provider`, `ai-orchestration`, `ai-observability`, `ai-infrastructure`, `ai-patterns`
- Meta domain categories: `meta-reviewing`, `meta-methodology`
- Infra domain categories: `infra-ci-cd`, `infra-platform`, `infra-config`
- Domain display names in wizard: "AI", "Methodology", "Infrastructure"

## Changed

- 20 skills moved from `api-ai` to dedicated `ai` domain with 5 categories
- 3 skills moved from `shared-meta` to `meta` domain (reviewing, cli-reviewing, research-methodology)
- 4 skills moved from `shared` to `infra` domain (docker, github-actions, cloudflare-workers, setup-env)
- 3 managed database skills recategorized from `api-baas` to `api-database` (upstash, vercel-kv, vercel-postgres)
- Domain order updated: Web, API, AI, Mobile, CLI, Infrastructure, Methodology, Shared
- Category count: 43 → 49
- All default stacks updated to reference new category/skill IDs
- E2E test infrastructure updated for new domain names

## Removed

- `api-ai` category (replaced by 5 `ai-*` categories)
- `shared-meta` category (replaced by `meta-reviewing`, `meta-methodology`)
- `shared-ci-cd` category (replaced by `infra-ci-cd`)
- `shared-infra` category (replaced by `infra-config`)
