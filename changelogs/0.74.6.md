# Release 0.74.6 (2026-03-13)

## Fixed

- **init**: Only show dashboard when a project CLI config (`.claude-src/config.ts`) exists — `hasIndividualPlugins` was causing false positives from Claude Code's own `.claude/settings.json`, showing the dashboard instead of the GlobalConfigPrompt
- **wizard**: Resolve custom skill categories from matrix provider via `getCategoryDomain()` instead of `isCategory()` static type guard — custom skills with auto-synthesized categories (like `engine-image`) were excluded from wizard pre-selection
- **edit**: Register marketplace before plugin installation, matching the existing logic in init — without this, `claude plugin install` fails when the marketplace hasn't been registered yet
