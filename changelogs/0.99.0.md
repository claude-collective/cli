# Release 0.99.0 (2026-03-30)

## Removed

- `config`, `config show`, `config path` commands and all associated tests
- `diff` command and all associated tests
- `outdated` command and all associated tests
- `diff` npm dependency (no longer needed)

## Added

- 12 new mutual-exclusion rule groups: documentation frameworks, desktop frameworks, React/Vue forms, BaaS platforms, authentication solutions, cache/KV providers, SQL databases, managed Postgres providers, i18n libraries, WebSocket variants
- Requirements for 15+ skills (framer-motion, error-boundaries, file-upload-patterns, image-handling, react-intl, tanstack-table, radix-ui, jotai, vue-i18n, storybook, docusaurus, vitepress, vuetify, tanstack-form, resend-react-email, clerk)
- React Native added to state library compatibility lists (zustand, redux-toolkit, mobx, jotai, react-query, swr)
- 6 new categories in built-in matrix: api-cms, api-commerce, api-framework, api-search, api-vector-db, meta-design
- 40+ new skills in built-in matrix
- New alternative groups: documentation frameworks, desktop frameworks, AI evaluation, SQL databases, React i18n
- Codex-keeper usage guide

## Changed

- Narrowed graphql-apollo compatibility to React-only (was multi-framework)
- Narrowed headless-ui compatibility to React-only (v2 dropped Vue support)
- Narrowed better-auth requirement to Drizzle only (skill teaches Drizzle adapter)
- Removed React from Expo compatibility (Expo implies React Native)

## Fixed

- Confirm step scroll viewport now accounts for border rows when calculating max scroll offset
- Scroll overflow and flexShrink only applied when content exceeds viewport
