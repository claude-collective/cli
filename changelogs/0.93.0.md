# Release 0.93.0 (2026-03-28)

## Fixed

- **Absolute localPath for local skills** — `localPath` on `ResolvedSkill` and `ExtractedSkillMetadata` changed from relative (e.g., `.claude/skills/my-skill/`) to absolute (e.g., `/home/user/project/.claude/skills/my-skill/`). Eliminates ENOENT errors in dual-scope edit when cwd differs from projectDir. Fixes D-152, D-92.
- **E2E dual-scope edit tests passing** — `dual-scope-edit-display` Test 1 and `dual-scope-edit-integrity` Test 7 no longer expected-fail

## Changed

- **Operations reorganized into domain subfolders** — 15 operation files moved from flat `operations/` into `source/`, `skills/`, `project/` subdirectories with subdirectory barrel files. Root barrel re-exports unchanged, so all consumer imports are unaffected. Resolves D-154.

## Removed

- **Dead lifecycle operations** — `executeInstallation` and `recompileProject` deleted (zero consumers; commands compose primitives directly)
